# KL
Windows向けクラシックタスクバー上ランチャー。
Windows10以前で出来ていた、ファイルやフォルダをタスクバーに設定してランチャーとして使えなかったので自作。

## 使い方
1. KL.exe をダブルクリックして起動します。
2. Windowsのタスクバー設定でKL.exeアイコンを隠さないように設定変更します。Windowsのバージョンによりますがタスクバー右クリック→タスクバーの設定→その他のシステムトレイアイコンから設定できます。
3. アプリのアイコンを右クリックして「メニュー登録フォルダを開く...」をクリックすると登録用フォルダが開きます。
4. エクスプローラ上でフォルダを右クリックドラッグアンドドロップして、表示させたいフォルダやファイルのショートカットを配置してください。


## 既知の問題
- ランチャー上のファイルを右クリックしたときのコンテクストメニューはOSの標準メニューを呼び出しているのですが、表示までの遅延が多いです。
- ランチャー上でのドラッグアンドドロップ等でファイル操作は出来ません
